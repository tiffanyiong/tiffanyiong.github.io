書的作者：張萌

# 讀書筆記總結

在我看來，人生效率手冊，是一本由許多不同的工具所組成的工具書，幫助你更加高效率的工作和學習。也許有些概念或詞你早已聽過了，比如說 "滿 10000 個小時就可以修煉成大師"、“21 天養成一個習慣 "、" 刻意練習 "、" 蕃茄工作法 "、" 目標管理 SMART 法則 "等等。我們先不管" 是否練習一千小時就能成大師 "或"21 天是否真的能養成一個習慣 " 這些問題，在讀本書的時候，我們更多讀到的是，作者如何使用這些工具幫助她高效的工作。下面分享幾個我在讀這本書的時候挺打動我的要點。

# 001 七個人物法 - 用於設定人生目標

不知道大家有沒有曾經迷惘過自己到底要成為怎麼樣的人？就算我渴望擁有別人的人生，也不知道怎麼努力？沒有方向也沒有目標怎麼辦？或者是，就算我早起了，也不知道要干嘛？
那個作者張萌自創的 "七個人物法" 可以幫你設立人生的目標，讓你知道你讓修煉甚麼硬本領（他們的核心競爭力，如時間管理、英語演講等）。

# 方法

首先寫下你第一個榜樣的名字 。這個人是你想要成為的人，而不是你的偶像。
然䈗分別寫下六個榜樣的名字 （分高，中，低寫）

  • 高： 也许终其一生也无法达到他们的高度
    （我自己寫下了比爾蓋茨，大概我努力一輩子也沒有用 Q_Q）
  • 中： 他们是我们用五到十年的时间就可以超过的人（超過現在的他）
  • 低：他們是你一年裡可以超越的人，這裡是可以寫身邊的人

# 今年目標

寫下來之後可以分別列出他們的三樣硬本領，最後你會得到 21 個硬本領。
這 21 個裡可能會有重覆，我們可以把重覆最多的三個硬本領列為今年的目標，成為你 "早起要干嘛？" 學習的方向。

# 002 時間管理

你有沒有試過拿著一個小本計劃一天要做的事情，可最後發現總是很難 follow? 寫下來的事情都沒有做到？根據《卓有成效的管理者》一書提到過管理自己的時間是分了三個步驟 ：記錄時間、管理時間、和統一安排時間。而作者在時間管理這一塊，建議使用的 "單點突破法" 也包括了這三點，接下來想寫下來分享自己的想法。

# 單點突破法 ：計劃 - 實施 - 總結 - 評估

單點突破法是一個閉環的結構：計劃 、實施 、 總結、評估 到再次計劃。

# 計劃

作者在全書中，其實對計劃給的建議不多，對我來說印象最深刻的是：在做計劃的時候不要安排得太滿，在第一件 跟 第二件事情的中間預留 30 分鐘緩冲時間，用來處理突發事情等。

# 實施 （包括紀錄時間）

很重要的一點是要學會紀錄時間，要知道你實際把時間花在哪了，才能做總結和評估。
作者紀錄時間的方法是：在微信中紀錄時間。她把自己幾個微信帳號拉到群裡，在完成一件事後，都會在群裡說一下，比如："已經離開會場"、"演講結束" 等。 這樣便可馬上紀錄真實的時間。

# 總結 - 評估

最後要做對比的工作。
把計劃表 和 紀錄的時間 做一個對比。可以問問自己：你計劃的跟你實施的時間是否一致？
要總結為甚麼沒有做到的原因。是因為能力不足嗎？還是有突發事情？
找出原因然後反思，才能幫助更好地計劃明天。

# 003 提高效率

# 蕃茄工作法

作者建議使用蕃茄工作法提升效率，在列出任務後，設定 25 分鐘的蕃茄鐘，心無雜念專注地執行你的任務。

# 運用目標 SMART 法則

“S：Specific，具体的。
M：Measurable，可衡量的。
-（ 量化一個目標！記住任何目標都是可衡量的，無法衡量的目標就等於沒有目標。）
A：Attainable，可实现的。

  • （制定好短时间内的可实现目标，完成之后，再去制定下一个短时间可实现的目标。）
    R：Relevant，相关的。
    -（意思就是目标分解的每一个小动作，都应该跟主目标相关）
    T：Time-Bound，有時限的。

# 004 小總結

這本書提醒了我的一點是，人類和其他動物其中最大的區別是："人類會制造 & 使用工具"。
在這本書，你會讀到關於作者自己原創的 "效率手冊" 的產品。也可以讀到她如何使用不同的方法 / 工具來自我管理。 推薦本書給迷惘的人們，也許你讀完之後，你會有了一個明確且可以努力的方向。
