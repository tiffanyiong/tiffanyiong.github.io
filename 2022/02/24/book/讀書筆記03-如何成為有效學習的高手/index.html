# 讀書筆記：如何成為有效學習的高手

書的作者：Cal Newport

# 學習的技巧

# 停止假用功，花的時間多不代表有效喔。

對於學生來說，完成的功課 ＝ 所費時間 x 注意力強度。
而假用功的特點就是注意力太低了，得花更多的時間，我們該學會加強注意力從而減少付出更多的時間。

# 提升效率的步驟

001 管理你的時間：5 分鐘清單法
需要工具： 日歷 ，一份單子（每天能帶上的，也可以從筆記本撕下的一張紙之類的）
方法：把要做的事情和其截止時間寫在日歷，並簡單在單子寫上每天的任務，再花 5 分鐘寫今天的安排。
重點：1）寫上時間。比如："11：00 - 12：30"： 上生物課。
2）不要安排太多
3）日程表上的時間並不是死的，可以自由變更任務

002 對抗拖延的建議

  1. 堅持做課並進展日誌 ：如果任務完成，寫上完成！未完成的話，標記下來並寫出原因
  2. 保證給身體補充能量 ：經常喝水！當能量低時反應遲鈍，容易拖延。喝水可以增加能量，作者的習慣是每 45 分鐘喝 5 小杯水
  3. 把棘手的任務當作大事 ：比如告訴所有你認識的人你要早起！知道的人愈多，你就愈難決定放棄不做。
  4. 形成常規 - 比如：每星期一、三、五的晚上 7 點準時運動 / 學習 / 寫作業。
  5. 提前計劃，不等到最後期限才做： 把事情堆在一天做完，你會感覺精力用盡的。

003 選擇適合的學習時間、地點和時長

  1. 學習時間愈早愈好：晚上的時間並沒有你想像的多注意力也不高，吃完飯開始學習，也就沒幾個小時，睡覺的慾望就來。晚上也是人們最想和朋友交往的時間。
  2. 找個安靜的地方，把自己隔離起來學習：
  3. 每次學習時長建議 40 - 50 分鐘（中間無休息），學習後休息 5-10 分鐘再重新學習。

004 做好筆記
對非技術類課程：使用 "問題 - 論據 - 結論" 的結構抓重點
對技術類課程：抓住問題的細節解釋，記越多越好。優先：1）記問題和答案 2）不懂就問 3）記例題的步驟 4）給步驟加注釋

005 避免思維疲勞

  1. 每天做一點點作業，不堆在一天裡做完
  2. 挑選必須讀的閱讀材料，老師給的不一定全部都要看
  3. 不會的題找老師 / 同學 / 朋友，進行小組討論

006 有效記憶
對非技術類課程：抽卡片記憶、大聲朗讀
對技術類課程：解題！如果你沒法確切解釋如何從問題到答案 ＝你沒有理解問題

# 如何高效寫作

論題：有關於某領域的一些言論，這個領域往往能激發人們很大的興趣
論點：指從這些言論中提煉出來的一個具體有新意的觀點

001 選題
必須及早動手！和老師交流、認真想想教材中哪些內容及引你 / 有疑問 / 爭論的地方。

002 找有價值的論點
做研究
分析評論性文章：把與論點想關的筆記都复習
研究性文章：必須找大量相關的資料，認真讀才能找到明確的論點

003 頭腦風暴

  1. 問自己： "我的觀點符合這次文章的要求嗎？" ，"涉及的內容有 specific 嗎？"，"會不會太簡單了？"
  2. 與老師交流自己想法

004 機器寫作法
研究性文章的核心是找資料，想要避免在收集資料時浪費時間的話，建議：反覆確認論點 > 找資料 > 复印 > 整理 > 加注釋

找資料的建議
資料一般分為：
一般性資料（整體性評論：如傳記、教材）
具體資料（如論證某觀點的期刊文章或書）

  1. 找在資料中的引用
  2. 使用期刊數據庫
  3. 請教圖書館理員，他們是可以幫你找資料的。

005 構思故事（組織）

  1. 寫作前:：把找到的資料都翻一篇，讓自己回憶所有相關信息
  2. 寫提綱 ： 建立論題提要，依次列出在文章中你要討論的問題

006 向高手請教
首選與老師交流，把提綱給他看（除了某些文章說明不多討論外都很樂意與你討論）再來和朋友交流

007 寫作技巧

  1. 多練習
  2. 參考文章寫作指導書
  3. 按照提網寫作：以提網為向導，填充適當內容

008 修改

  1. 優化論證結構：一次讀一段，條理混亂要改。觀點重覆就刪，細節不足補上兩句
  2. 朗讀出來看看有沒有小錯誤
  3. 改語法！！！
  4. 最後檢查～
