# 曾國藩家書（精選白話文紀念版）

如果沒有數錯的話，這本《曾國藩家書》只有 153 通家書，然而這是我第一次讀別人的家書，第一次讀初步了解曾國藩先生的故事。家書裡除了看到大部份給家人們（尤其是給他弟弟們）的訓誡外，更多的是關於他的爲人處世之道。關於他生平等事蹟，希望在讀下一本書《曾國藩的正面與側面》的時候可以有更深入的了解！

書的編譯者把曾國藩先生的家書翻釋成白話文，並且分成十個部份（十個主題），包括學習、修身、家族、教育、職場、領導力、用人、財富、社交、和養生。下面，摘錄我比較印象深刻的文字（白話文）。

# 第一篇：學習

我們讀書只為兩件事：一、為了提高道德修養，這需要講求誠意、正心、修身、齊家的道理，如此才能做到不負一生；二、為了修業保身，這需要練習記誦詞章的技巧，如此才能做到自立自衛。

士人讀書的時候，要講求以下三點：一、要有志氣；二、要有見識；三、要有恒心。有志氣的人不會甘居下游；有見識的人明白學無止境，不會因一點小成就而自我滿足，不會有河伯觀海、并蛙窺天等無知想法；有恒心的人不會有做不成的事情。

讀經要有耐心，有一句不理解，那就不看下一句；今天不理解，明天接著再讀；今年不理解，明年接著再讀，直到自己弄明白為止。這就是有耐心的表現。

我希望弟弟們不要過於在乎是不是有科名，而要首先注重孝和悌，其次才是寫一手不朽的文章。弟弟們如果真的能夠自立的話，那就應當抓住大的、長遠的目標，而不要只是追求進學這一件事情。

我原本以為離開家庭那種狹小的天地，而和省城裡那些比自己強的人相處，進步一定非常大。這兩年以來，看書不多，詩文方面則毫無長進，因此不能歸咎於地方的局促。

我所期待弟弟們做到的，就是這樣，總離不開 “立志” “有恆” 這四個字。

# 第二篇 修身

要取得進步的話，首要的就是要去掉驕傲的習氣。本來肚子裡就沒有什麼東西，又夜郎自大，這是最容易壞事的。

那些因為懼怕他人而不敢妄自進行議論的人，是謹慎謙和的人；而那些喜歡諷刺批評他人短處的人，則是驕傲的人。

要戒掉 “驕” 字，首要的是不輕易議論或譏笑他人；而要戒掉 “惰” 字，首要的則是不要起床太晚。

我們都很難處世，我常常為此而感到焦慮。也許心裡常常抑鬱總是怨天尤人的人，非但不能涉世，而且也不好提高品行和保養身體。

我家中的後代子弟，一點都沒經歷過那種艱難困苦的生活，所以變得自視甚高，口氣也非常硬，整天呼喚奴婢，以致最後養成不好的習氣。他們身上的這股驕傲之氣，明明已經病入膏肓，但絲毫沒有覺察，對此我深感憂慮。

造成戰爭失敗的原因很多，有客觀的，也有主觀的。在主觀原因中，必然少不了 “傲” 或 “惰”。大家族衰敗的原因也不過如此，不是傲就是惰，二者必有一個。
眼睛看到千里之外，但卻看不到自己的睫毛。

在 “孝悌” 兩個字上，你盡一分力，就會得到一分學問；你盡十分力，就會得到十分學問。現在的人們讀書只有一個目的，那就是通過科舉獲得功名。他們似乎一點都不關心孝悌倫紀這些大道理，甚至在他們看來，這些與讀書做學問根本就沒什麼關係。殊不知，古書中記載的歷代聖賢所作的文章，就是要闡明這個道理。如果平時你的行為符合孝悌，那就算是你的筆下寫不出來又有什麼關係呢？如果平時總是違背孝悌做事，那就算是你寫的文章再精彩，也只能被稱為名教中的罪人。

# 第三篇 家族

我和沅弟討論治家之道時，我們都認為以星岡公為準繩。總體來說，治家之道有八字訣，即 “書、蔬、魚、豬、早、掃、考、寶”。
家庭的敗落，隱藏一個 “漏” 字；個人的失敗，一個 “逸字”；遭人嫌棄，疑一個 “驕” 字。
要離開驕傲之氣，想去的就是不要輕易地嘲笑別人；要懶惰的主動之舉，目的地的就是每天要早起。
勤” 字為生動之氣，“儉” 字為收斂之氣。家人能做到這兩個字，家運就一定會興旺。

# 第四篇 教育

孔子說：“想要自立，先使別人自立；想要發達，先讓別人發達起來”；孟子說：“凡是行為得不到預期的效果，都應該反過來檢查自己。
你讀書的記憶力一般，但你對此不必過多顧慮。你真正應該擔心的有兩點：一是做事怕沒有恆心；二是怕隨意粗略地看書，從來沒有理解其中的內容。
不管年齡有多大，不管事情的難易程度如何，只要能夠一直堅持下去，每一件事情都會像種植樹木每日都在成長，只是你沒有發覺。
人們之所以多少會顧及體面，就是因為每個人都希望得到他人的敬重。如果一個在人前完全暴露了自己的驕傲、懶惰、惡劣的言行，那他就會沒有一點顧忌，乾脆就把所有的廉恥之心都拋掉，也不再顧及臉面，下定決心與正直的人作對。到這個時候，這樣的人也無可救藥了。
什麼樣的人才是君子呢？那些勤儉自持，習慣勞苦生活，既能享受安樂，又能適應儉約的人，就是真正的君子。
人的氣質本來是天生的，早就已經註定了的，很難改變，只有讀書才是重新塑造氣質的途徑。古代那些精通相面術的人，都認為讀書能改變一個人的骨相。但要真正改變骨相，就一定要先有堅定不移的志向。

# 第五篇 職場仕途

我教導子弟要遵守 “八本” 和 “三致祥”。“八本” 指的是：讀古書以訓話為本，作詩文以聲調為本，養親以得歡心為本，養生以少惱怒為本，立身以不妄語為本，治家以不晚起為本，為官以不要錢為本，行軍以不擾民為本。
“三致祥” 指的是：孝致祥、勤致祥和恕致祥。我的父親竹亭公教導後輩，以孝為重點。年少的時候敬親，晚年的時候愛親，都出於一顆至誠的孝心。

# 第六篇 領導力

當與他人接觸周旋的時候，如果不能做到以誠相待，那就不會感動他人。但只有誠意，而沒有在語言文字的表達上好好下工夫打動人，那滿腔的誠意也無從表達。《禮》中有這樣一句話：“沒有文采，行而不遠。” 其實表達的也是這個意思。
一是天懷淡定，不要強求速效；二是謹防有敵人的援兵到來，以免城內外的敵人聯合猛撲，一定要穩妥慎重地做好防禦工作。

# “第七篇　用人”

一般來說，軍氣可以聚，但不可以散；可以憂危，但不可以太過愉悅。軍營中人多，就會有愉悅的氛圍。這種情況下，士氣就會慢慢消散。軍營雖然很多，但能倚靠的只有一兩個營；軍營里人雖然很多，但能倚靠的只有一兩個人。

一般來說，成為將才需要具備四個方面的素質：第一能知人善任，第二善於偵察敵情，第三在敵陣面前有足夠的膽識，第四營務要整齊。

一個人走一百里路，當走到九十里的時候，也只能算走了一半路程。

精神是一種越用越好用的東西，不能因為自己身體虛弱而過分愛惜它；智慧則是一種越是苦越閃光的東西，不能因為一時遭遇挫折就倉皇自棄。

對辦大事的人來說，多選接替人手是首要的事情。選不到滿意的人，那就退而求其次，然後再慢慢地教育培養。

現在保薦人才，也有很多讓人為難的地方。有的時候保薦，旁人並不認同你的看法，反而會連累被保薦之人；有的時候，被保薦的人不因此而感恩於你，反而會給自己帶來仇恨。

天下的事理人才，我們不知道、沒有想到的有很多，千萬不能有自滿的情緒。用人不輕率、用人冗繁，為人不自滿，能做到這兩點，就一定能避免出現過失，也一定不會失去自己的名聲。這一點請牢記在心。

養身的方法大概包括五個方面：第一是在睡眠和飲食上有科學的規律；第二是不要發怒；第三是要節欲；第四是每天睡覺之前要洗腳；第五是兩餐飯後，要各走三千步。養生以少惱怒為本，也就是我 “八本” 中說的 “養生以少惱怒”。睡眠、飲食有規律，以及睡覺前洗腳的事情，星岡公一直堅持了四十年。我也堅持了七年，這些日子我在飯後都要走上三千步，從此以後，絕對不會間斷。
