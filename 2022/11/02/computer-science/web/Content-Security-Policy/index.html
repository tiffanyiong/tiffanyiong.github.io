在開發的時候遇到好多次 resource 被 blocked 掉的問題，很多時候都是因為沒有把 resource 加入在 "Content Security Policy" 的 config 裡面。
Inspect 的時候可以看到會出現類似下圖的 Error

只要在 config 裡的 Content Security Policy 的 directive "script-src" 裡面就可以了。
